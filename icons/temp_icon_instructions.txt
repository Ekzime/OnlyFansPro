Инструкции для создания иконок:

1. Откройте create_icons.html в браузере (должно было открыться автоматически)
2. Нажмите на ссылки для скачивания:
   - icon16.png
   - icon32.png  
   - icon48.png
   - icon128.png
3. Сохраните их в папку icons/

АЛЬТЕРНАТИВНО - для быстрого тестирования:
Можете временно использовать любые 16x16, 32x32, 48x48, 128x128 PNG файлы
и переименовать их соответственно.

После создания иконок расширение будет готово для тестирования!
